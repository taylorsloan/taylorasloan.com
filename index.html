Special Delivery

Special Delivery is a motion-activated GPS-enabled tracking device based on the Particle Electron microcontroller. I designed the device designed to be deployed in a bait package to catch mail thieves. It uses an on-board GSM modem to relay its position in real-time. Location and status updates are provided via email. The device can be charged wirelessly and it lasts for months on a single charge.

PH Doser

This gadget was designed to provide an automated solution for dosing water to achieve a desired Ph level. The design uses a Ph probe that polls for the current Ph level. If it is beyond the desired scope, acids or bases will be pumped in using peristaltic pumps. It features a full menu with settings to set Ph thresholds, different running modes, and calibrations. The code was finished and tested but the dosing apparatus was never finished. 

Samsung S7 Car Smart Dock

This is my own 3D printed Samsung S7 Edge Car docking station with an integrated wireless charging coil. I wrote a script to start Android Auto and a dashcam app when my phone detects that it is in the docking station. The phone returns to normal operation upon removal.
